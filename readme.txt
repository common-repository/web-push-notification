== Description ==

* <strong>Boost</strong> your readers engagement and send a smart <strong>Web Push Notification</strong> to your users each time you have new posts even if user is not surfing into your web site.

Web Push Notification works on chrome (Safari, Firefox coming soon).

<h4>Features</h4>
- Install and Activation Web Push Notification.
- Register your web site.
- 03 keys will be generated for you, the private key is your ONLY WAY to send any request to our server via our secure api.
- Public key: -
- Private key: PLEASE KEEP IT SECURE
- Testing key: PLEASE KEEP IT SECURE
- Copy and save them into Web Push Notification admin panel
- Set your action message
- You can see a Demo how your posts will look like any time
- Start sending notifications to your users directly from post edit page
- Check all waiting posts any time you want.

- By using this plugin you agree that, during any interaction with our server, your email or your blog name or your blog url will be sent to our server.

If you have any questions about our Privacy, please contact us.
	
== Installation == 
* In your Admin, go to menu Plugins > Add
* Search for Web Push Notifications
* Click to install
* Activate the plugin
* Register your web site using the new Web Push Notification page under Settings menu.
* Start sending notifications to your users directly from post edit page.
